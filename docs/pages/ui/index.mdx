# @starknet-io/get-starknet-ui

## Installation

Install Get Starknet using your favourite package manager.

:::code-group

```bash [npm]
npm install @starknet-io/get-starknet-ui@next
```

```bash [pnpm]
pnpm add @starknet-io/get-starknet-ui@next
```

:::
