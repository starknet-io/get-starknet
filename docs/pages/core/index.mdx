# @starknet-io/get-starknet-core

This package provides the core functionality. It does not include any UI components and as such it does not depend on React.

## Installation

Install Get Starknet using your favourite package manager.

:::code-group

```bash [npm]
npm install @starknet-io/get-starknet-core@next
```

```bash [pnpm]
pnpm add @starknet-io/get-starknet-core@next
```

:::
