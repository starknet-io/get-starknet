# getStarknetChainId

Get the 0x-prefixed chain ID from a Starknet chain ID.

## Usage

```ts
import { getStarknetChainId } from "@starknet-io/get-starknet-core";

getStarknetChainId("starknet:0x1234");
````

## Arguments

 * `chain: string` - The Starknet chain ID.

## Returns

 * `ChainId` - The 0x-prefixed chain ID.
