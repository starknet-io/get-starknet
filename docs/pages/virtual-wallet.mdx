# Virtual Wallets

The `@starknet-io/get-starknet-virtual-wallet` package provides support for so
called "virtual wallets" based on the EIP-1193 provider.

## EIP-1193 Adapter

The package exports a type `EIP1193Adapter` that models how virtual wallets are
created.

```ts
export type EIP6963ProviderInfo = {
  rdns: string
}

export type EIP1193Adapter = (
  info: EIP6963ProviderInfo,
  provider: EIP1193Provider,
) => WalletWithStarknetFeatures | null
```

This interface is used by [discovery](./discovery) to detect and create virtual
wallets.

## MetaMask

The package provides the implementation for the MetaMask virtual wallet.

Given a EIP-1193 provider, the `MetaMaskVirtualWallet` class is used to lazily
load the MetaMask Snap on first use. The class implements the
[Starknet Wallet Standard](./wallet-standard) and can be used anywhere a
Starknet wallet is expected.

```ts
import { MetaMaskVirtualWallet } from "@starknet-io/get-starknet-virtual-wallet/metamask"

const wallet = new MetaMaskVirtualWallet(provider)
```
