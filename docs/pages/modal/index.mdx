# @starknet-io/get-starknet-modal

This package provides a headless UI component to connect a Starknet wallet to your application. It allows you to customize the UI and integrate it seamlessly with your existing design system.

## Installation

:::code-group

```bash [npm]
npm install @starknet-io/get-starknet-ui@next
```

```bash [pnpm]
pnpm add @starknet-io/get-starknet-ui@next
```

:::
